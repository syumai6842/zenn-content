# 日次公開の手順

セッション内で以下を実行する。`claude -p` は使わない（別枠課金を避ける）。

## 使い方

対話セッションで以下のように指示する:

```
今日のZenn本3本を作って
```

または特定テーマを指定:

```
以下の3テーマでZenn本を作って
1. ...
2. ...
3. ...
```

## 内部動作

1. `.claude/outputs/research/zenn-theme-queue.md` からテーマを3本取り出す
2. 各テーマについて `zenn-publisher` agent を起動（3本並列）
3. 各agentが リサーチ→執筆→安全チェック→git push を自律実行
4. 完了後、テーマキューから使用済みテーマを消す

## テーマキューの補充

キューが10本を切ったら自動でresearcherを起動してテーマ候補を追加生成する。
